# Design System Plan

## Overview

Multi-framework UI component library enabling consistent UX across the entire stack - Astro blog, Directus CMS, and future applications.

## Goals

1. Framework-agnostic components via DaisyUI
2. Published npm package for easy consumption
3. Storybook documentation for all components
4. Multi-framework support (React, Vue, Svelte)

## Current State

- React components established
- Storybook 8 configured
- npm package published
- GitHub Pages hosting Storybook
